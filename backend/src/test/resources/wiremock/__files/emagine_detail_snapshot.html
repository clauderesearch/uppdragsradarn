SharePoint Expert (Ledningssystem)

#211886
Start
ASAP
Duration
7-9 months

emagine söker efter en SharePoint Expert till vår kund i Stockholm.

Start: Asap

Slut: Höst -25
Work location: Stockholm, Hybrid setup

Kompetens:

Teknisk kompetens i SharePoint för att bygga upp ett klickbart ledningssystem (3-4 nivåer)

Projekt beskrivning:

Vi ska flytta vårt ledningssystem från servrar till SharePoint och vill göra det enklare att hitta det man söker. Jag vill ha flera ingångar, alltså tex via processer, projekttyp och specialistområden, så att ledningssystemet upplevs som det stöd det ska vara. Vi kommer med input kring struktur/visualisering men kan inte bygga detta själva i SharePoint, där behöver vi hjälp. Oklart om vi kan bygga det vi vill, antal nivåer, etc med funktionaliteten som finns inbyggd i SharePoint eller om vi behöver köpa något system – det behöver vi expertens hjälp med att utreda.

Arbetsbeskrivning:

Rent praktiskt hjälpa oss att bygga upp vårt ledningssystem i SharePoint/sätta igång oss och finnas som bollplank.

Skallkrav:

Ledningssystem enl ISO (helst alla tre standarder), inkl visualisering

Sharepoint

 

Personlighet:

Samarbetsinriktad och lösningsorienterad
